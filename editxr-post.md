# Editxr is a text editor made with an LLM

This is a text editor I made with Claude. It is built in Swift; I started with an existing library and iterated from there.

It has wordwrap (but you can turn it off easily). It renders **markdown** the way I expect it to. It is *TUI*, which makes it interesting, because generally I like editing markdown files this way.

This post is written in Editxr; the name is inspired by its use today to be inclusive.

## How the idea came about

I always wanted to make a minimalist text editor; I have been a user of VI and VIm for more than two decades.


# Integración con LLMs

No me convence la forma de edición existente, me gusta editar en forma minimalista, pero teniendo las herramientas disponibles en unos pocos shortcuts. Una de las herramientas mas importantes del momento es el acceso a un LLM, para poder reescribir, editar y expandir secciones.

Las ediciones deben quedar integradas, pero no mezcladas, por lo cual las ediciones se debe poder ver, con un modo ver o no. Esto por el momento no está implementado, solo se funde en un solo texto.
